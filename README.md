# Bloomberg Terminal CLI

A Bloomberg Terminal-like CLI application built with Textual.

## Installation

```bash
pip install -e ".[dev]"
```

## Usage

```bash
bloomberg
# or
python -m bloomberg_terminal
```

## Commands

| Command | Action |
|---------|--------|
| `Q AAPL` | Get quote for AAPL |
| `GP AAPL` | Price chart for AAPL |
| `FA AAPL` | Fundamentals for AAPL |
| `N` | News feed |
| `W` | Show watchlist |
